# Onboarding Images

This directory contains the images used in the onboarding screens.

## Required Images

Please add the following images (280x280px recommended):

1. **inventory.png** - Image representing inventory management
   - Should show products, stock levels, or warehouse themes
   - Used in the first onboarding slide

2. **sales.png** - Image representing sales and transactions
   - Should show invoices, transactions, or customer interactions
   - Used in the second onboarding slide

## Image Specifications

- Format: PNG (preferred) or JPG
- Recommended size: 280x280 pixels
- Transparent background preferred for PNG
- Keep file sizes under 500KB for optimal performance

## Usage

These images are referenced in:
- `apps/native/app/onboarding.tsx`

The images will have a bubble-up animation effect applied automatically.
